Student Attendance Management System-
This is a web-based Student Attendance Management System developed using Google AI Studio. The application provides an easy and efficient way to record, manage, and track student attendance digitally.

Features:
Add and manage student records
Mark daily attendance
View attendance history
Track attendance status
User-friendly interface
Fast and efficient attendance management

Technologies Used:
HTML
CSS
JavaScript
Google AI Studio

Purpose:
The project aims to simplify the traditional attendance process by providing a digital solution that reduces manual work, improves accuracy, and helps educators maintain attendance records efficiently.

Future Enhancements:
Student login and authentication
Attendance analytics and reports
Export attendance data to Excel/PDF
Email notifications for low attendance
Cloud database integration
